6.2 Subspaces and Spanning Sets

In this section, we revisit the notions of subspace, linear combination and spanning set from the perspective of abstract vector spaces.

Vector Subspaces

Let $V$ be a vector space and $U\subseteq V$ a subset. We say that $U$ is a (vector) subspace of $V$ if it satisfies the following three conditions:
  1. $\bf{0}\in U$.
  2. If $\bf{u}$ and $\bf{v}$ are in $U$, then $\bf{u}+\bf{v}\in U$.
  3. If $\bf{u}\in U$ and $a\in \RR$ is any number, then $a\bf{u}\in U$.

Observe that if $V$ is a vector space and $U$ is a vector subspace of $V$, then $U$ is also a vector space itself when endowed with the addition and scalar multiplication that come from $V$.

If $V$ is a vector space, then $\{\bf{0}\}$ and $V$ are both subspaces of $V$. Any subspace of $V$ that is different from $\{\bf{0}\}$ and $V$ is called a proper subspace.

Let us see some example of vector subspaces in the vector spaces that we defined in the previous section.

Suppose $V$ is any vector space and $\bf{v}\in V$ is a vector. We define the line generated by $\bf{v}$ as the subset \[ \RR\bf{v}=\{ a\bf{v} \mid a\in \RR \} . \] Show that $\RR\bf{v}$ is a vector subspace of $V$.
Solution.

We first check that $\bf{0}\in \RR\bf{v}$. This is true because $\bf{0}=0\bf{v}$, so $\bf{0}\in V$.

Now, suppose we are given two vectors $\bf{u}_1,\,\bf{u}_2\in V$. This means that $\bf{u}_1=a_1\bf{v}$ and $\bf{u}_2=a_2 \bf{v}$ for some real numbers $a_1,\,a_2\in \RR$. Therefore, \[ \bf{u}_1+\bf{u}_2 = a_1 \bf{v} + a_2 \bf{v} = (a_1+a_2)\bf{v}, \] so $\bf{u}_1 + \bf{u}_2 \in \RR\bf{v}$ as well.

Finally, suppose $\bf{u}\in \RR\bf{v}$ and $a\in \RR$ is a scalar. Since $\bf{u}\in \RR\bf{v}$, there exists a number $b\in \RR$ such that $\bf{u}=b\bf{v}$. Thus, \[ a\bf{u}=a(b\bf{v})=(ab)\bf{v}, \] which implies that $a\bf{u}\in \RR\bf{v}$.

In conclusion, $\RR\bf{v}$ is a subspace of $V$ because it satisfies the three requirements.

Let $A$ be an $n\times n$ matrix. Then we define the following subset of $\bf{M}_{nn}$: \[ U = \big\{ X\in \bf{M}_{nn} \mid 2AX=5XA \big\}. \] Show that $U$ is a subspace of $\bf{M}_{nn}$.
Solution.

Firstly, the zero matrix $0$ is in $U$ because $2A0=20A=0$.

Assume $X$ and $Y$ are matrices in $U$. Then $X+Y\in U$ as well because \[ 2A(X+Y)=2AX+2AY=5XA+5YA=5(X+Y)A. \]

To finish, we note that if $X\in U$ and $a\in \RR$, then $aX\in U$ since \[ 2A(aX)=a(2AX)=a(5XA)=5(aX)A. \]

We thus conclude that $U$ is a vector subspace of $\bf{M}_{nn}$.

The subset \[ U=\{ p(x)\in \bf{P} \mid p(1)=0 \} \] is a vector subspace of the vector space $\bf{P}$ of all polynomials.
Solution.

The zero polynomial $0(x)$ is in $U$ since $0(1)=0$.

Now, let $p(x)$ and $q(x)$ be two polynomials in $U$. The sum $(p+q)(x)$ is also in $U$ because \[ (p+q)(1)=p(1)+q(1)=0+0=0. \] Similarly, if $a\in \RR$ is a scalar then $(ap)(x)\in U$ since \[ (ap)(1)=ap(1)=a0=0, \] so we deduce that $U\subseteq \bf{P}$ is a vector subspace.

The set $\bf{P}_n$ of all polynomials of degree at most $n$ is a subspace of $\bf{P}$.
The set $\bf{D}[a,b]$ of all differentiable functions on $[a,b]$ is a vector subspace of $\bf{F}[a,b]$.

Linear Combinations and Spanning Sets

We now extend our notions of linear combination and spanning set to abstract vector spaces.

Let $V$ be a vector space and $\bf{v}_1,\,\bf{v}_2,\,\dots,\,\bf{v}_k\in V$. A vector $\bf{v}\in V$ is a linear combination of the vectors $\bf{v}_1,\,\bf{v}_2,\,\dots,\,\bf{v}_k$ if there exist scalars $a_1,\,a_2,\,\dots,\,a_k\in \RR$ such that \[ \bf{v}=a_1 \bf{v}_1+a_2\bf{v}_2 + \cdots + a_k\bf{v}_k. \] The set of all linear combinations of $\bf{v}_1,\,\bf{v}_2,\,\dots,\,\bf{v}_k$ is called the (linear) span of the set $\{\bf{v}_1,\bf{v}_2,\dots,\bf{v}_k\}$. More explicitly, the span of $\{\bf{v}_1,\bf{v}_2,\dots,\bf{v}_k\}$ is the set \[ \vecspan \{\bf{v}_1,\dots,\bf{v}_k\} = \{ a_1 \bf{v}_1 + \cdots +a_k \bf{v}_k \mid a_1,\dots,a_k\in\RR \}. \] Moreover, if $V=\vecspan\{\bf{v}_1,\dots,\bf{v}_k\}$, we say that $\{\bf{v}_1,\dots,\bf{v}_k\}$ is a spanning set for $V$.
\[ \bf{M}_{23} = \vecspan\left\{ \begin{bmatrix} 1 & 0 & 0 \\ 0 & 0 & 0 \end{bmatrix}, \begin{bmatrix} 0 & 1 & 0 \\ 0 & 0 & 0 \end{bmatrix}, \begin{bmatrix} 0 & 0 & 1 \\ 0 & 0 & 0 \end{bmatrix}, \begin{bmatrix} 0 & 0 & 0 \\ 1 & 0 & 0 \end{bmatrix}, \begin{bmatrix} 0 & 0 & 0 \\ 0 & 1 & 0 \end{bmatrix}, \begin{bmatrix} 0 & 0 & 0 \\ 0 & 0 & 1 \end{bmatrix} \right\}. \]
Solution.

Observe that if we take any matrix \[ A = \begin{bmatrix} a & b & c \\ d & e & f \end{bmatrix}\in \bf{M}_{23}, \] we can rewrite it as \[ \begin{aligned} A ={}& a \begin{bmatrix} 1 & 0 & 0 \\ 0 & 0 & 0 \end{bmatrix} +b \begin{bmatrix} 0 & 1 & 0 \\ 0 & 0 & 0 \end{bmatrix} +c \begin{bmatrix} 0 & 0 & 1 \\ 0 & 0 & 0 \end{bmatrix} +d \begin{bmatrix} 0 & 0 & 0 \\ 1 & 0 & 0 \end{bmatrix} +e \begin{bmatrix} 0 & 0 & 0 \\ 0 & 1 & 0 \end{bmatrix} +f \begin{bmatrix} 0 & 0 & 0 \\ 0 & 0 & 1 \end{bmatrix}, \end{aligned} \] so it is indeed a linear combination of the above matrices.

More generally, we have:

$\bf{M}_{mn}$ is spanned by the set of all matrices that have exactly one entry equal to $1$ and the remaining ones equal to $0$.
Since every polynomial of degree (at most) $n$ is of the form \[ a_0+a_1 x + \cdots + a_n x^n \] for scalars $a_0,\,a_1,\,\dots,\,a_n\in \RR$, it follows that \[ \bf{P}_{n}=\vecspan\{1,x,x^2,\dots,x^n\}. \]

We now restate the span theorem from the previous chapter in terms of general vector spaces.

Suppose $\bf{v}_1,\,\dots,\,\bf{v}_k$ are vectors in a vector space $V$ and consider $U=\vecspan \{\bf{v}_1,\dots,\bf{v}_k\}$.
  1. $U$ is a vector subspace of $V$.
  2. If $W\subseteq V$ is another subspace and $\bf{v}_1,\dots,\bf{v}_k\in W$, then $U\subseteq W$.
Proof.

The second statement is proved exactly in the same way as in the case of $\RR^n$, so we only focus on proving that $U$ is a subspace of $V$.

The zero vector $\bf{0}\in U$, because we can write \[ \bf{0}=0\bf{v}_1+\cdots+0\bf{v}_k. \]

Now, suppose that $\bf{u},\,\bf{v}\in U$ are two vectors. From the definition of $U$, we have that both $\bf{u}$ and $\bf{v}$ are linear combinations of $\bf{v}_1,\,\dots,\,\bf{v}_k$, say, \[ \bf{u}=a_1 \bf{u}_1 + \cdots + a_k \bf{v}_k, \qquad \bf{v}=b_1 \bf{u}_1 + \cdots + b_k \bf{v}_k. \] Then we have \[ \bf{u}+\bf{v}= (a_1+b_1)\bf{v}_1 + \cdots + (a_k + b_k)\bf{v}_k\in U. \] Similarly, $a\bf{u}\in U$ for any real scalar $a$, because \[ a\bf{u}=(aa_1)\bf{v}_1+\cdots+(aa_k)\bf{v}_k. \] Thus, $U$ is a vector subspace of $V$.

Let us see some applications of the span theorem.
Show that the polynomials \[ p_1(x)=1+x^2, \quad p_2(x)=x-3, \quad p_3(x)=-2 \] span $\bf{P}_2$.
Solution.

Since $\bf{P}_{2}=\vecspan \{1,x,x^2\}$, it suffices because of the span theorem to prove that the polynomials $1$, $x$ and $x^2$ are linear combinations of $p_1(x)$, $p_2(x)$ and $p_3(x)$. But this is true because \[ \begin{aligned} 1&=-\frac{1}{2}p_3(x), \\[1em] x&= p_2(x)-\frac{3}{2}p_3(x), \\[1em] x^2&= p_1(x)+\frac{1}{2}p_3(x). \end{aligned} \] Thus, $\bf{P}_2=\vecspan\{p_1(x),p_2(x),p_3(x)\}$, as desired.

If $\bf{u},\,\bf{v}\in V$ are two vectors in a vector space, show that \[ \vecspan\{\bf{u},\bf{v}\}=\vecspan\{\bf{u}+\bf{v},-2\bf{v}\}. \]
Solution.

Let us start by proving that \[ \vecspan\{\bf{u},\bf{v}\}\subseteq\vecspan\{\bf{u}+\bf{v},-2\bf{v}\}. \] By the span theorem, it is enough to show that $\bf{u}$ and $\bf{v}$ both lie in $\vecspan\{\bf{u}+\bf{v},-2\bf{v}\}$. To see this, note that \[ \begin{aligned} \bf{u}&=(\bf{u}+\bf{v})+\frac{1}{2}(-2\bf{v}),\\[1em] \bf{v}&=-\frac{1}{2}(-2\bf{v}) \end{aligned} \] are both in $\vecspan\{\bf{u}+\bf{v},-2\bf{v}\}$, so the above inclusion holds.

We now have to guarantee that \[ \vecspan\{\bf{u}+\bf{v},-2\bf{v}\}\subseteq\vecspan\{\bf{u},\bf{v}\}, \] but this is immediate from the fact that $\bf{u}+\bf{v}$ and $-2\bf{v}$ are linear combinations of $\bf{u}$ and $\bf{v}$.

We thus conclude that \[ \vecspan\{\bf{u},\bf{v}\}=\vecspan\{\bf{u}+\bf{v},-2\bf{v}\}, \] as desired.

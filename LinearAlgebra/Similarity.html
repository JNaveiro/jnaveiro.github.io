5.5 Similarity and Diagonalization

In this section we revisit diagonalization of square matrices using the language of vector subspaces and bases.

Similar Matrices

Suppose $A$ and $B$ are square matrices of the same size. We say that $A$ and $B$ are similar (denoted $A\sim B$) if there exists an invertible matrix $P$ such that \[ B=P^{-1}AP. \]

The similarity relation satisfies the following three properties:

  1. $A\sim A$ (because $A=I^{-1}AI$).
  2. If $A\sim B$, then $B\sim A$ (because $B=P^{-1}AP$ with $P$ invertible implies $A=PBP^{-1}$).
  3. If $A\sim B$ and $B\sim C$ then $A\sim C$ (because $B=P^{-1}AP$ and $C=Q^{-1}BQ$ for invertible matrices $P,\,Q$ implies $C=(PQ)^{-1}A(PQ)$, with $PQ$ invertible).
It is common to say because of this that similarity is an equivalence relation.

With this language, we can restate notion of diagonalizability as follows: A square matrix $A$ is diagonalizable if it is similar to a diagonal matrix.

If $A$ is a diagonalizable square matrix, then every matrix that is similar to $A$ is also diagonalizable.
Solution.

If $A$ is diagonalizable, then $A\sim D$ for some diagonal matrix $D$. If $B$ is a square matrix such that $B\sim A$, then $B\sim D$ from the above properties, so $B$ is also diagonalizable.

If $A,\,B$ are square matrices and $A\sim B$, show that
  1. $A^{T}\sim B^{T}$.
  2. If $A$ is invertible, then $B$ is invertible and $A^{-1}\sim B^{-1}$.
  3. $A^{k}\sim B^{k}$ for all $k=1,2,\dots$
Solution.

Since $A\sim B$, we know that there exists an invertible matrix $P$ such that $B=P^{-1}AP$. We will use this fact to show that the three above items are true.

Firstly, by transposing the identity $B=P^{-1}AP$, we obtain $B^{T}=P^{T}A^{T}(P^{-1})^{T}=P^{T}A^{T}\big(P^{T}\big)^{-1}$, which shows that $A^{T}$ and $B^{T}$ are similar matrices.

Secondly, since $B=P^{-1}AP$ and both $A$ and $P$ are invertible, it follows that $B$ is invertible as a product of invertible matrices. Moreover, by inverting the previous equation we get $B^{-1}=P^{-1}A^{-1}P$, so $A^{-1}$ and $B^{-1}$ are similar matrices.

Finally, if we take the equation $B=P^{-1}AP$ and square both sides, we obtain $B^{2}=P^{-1}APP^{-1}AP=P^{-1}A^{2}P$, so $A^{2}$ and $B^{2}$ are similar matrices. For higher powers, the argument is the same.

It turns out that similar matrices will share many properties. For instance, if $A\sim B$, then $\det(A)=\det(B)$. This is because if $B=P^{-1}AP$ for some invertible matrix $P$, we get \[ \det(B)=\det\big(P^{-1}\big)\det(A)\det(P)=\frac{1}{\det(P)}\det(A)\det(P) = \det(A). \] There is another function that, while we will not be using it much throughout the course, it appears in several contexts:

If $A$ is an $n\times n$ matrix, then the trace of $A$ (denoted $\tr(A)$) is defined as the sum of the elements in its main diagonal. Explicitly, if $A=[a_{ij}]$, then $\tr(A)=a_{11}+a_{22}+\dots+a_{nn}$.

For example: \[ \tr \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \\ 7 & 8 & 9 \end{bmatrix} = 1+5+9 = 15. \] One can show by direct computation that \[ \tr(AB)=\tr(BA) \quad \textsf{for all square matrices } A,\,B. \]

If $A$ and $B$ are similar matrices, then they have the same determinant, trace, characteristic polynomial and eigenvalues.
Proof.

We already showed that $A$ and $B$ have the same determinant, so we have to focus on the other three items. As $A$ and $B$ are similar, we can write $B=P^{-1}AP$ for some invertible matrix $P$. Taking the trace of the above equation we get \[ \tr(B)=\tr\big(P^{-1}AP\big)=\tr\big(APP^{-1}\big)=\tr(A). \] Let us now find the characteristic polynomial of $B$: \[ \begin{aligned} c_{B}(x)&=\det(xI-B)=\det\big(x\big(P^{-1}IP\big)-P^{-1}AP\big) = \det\big(P^{-1}(xI-A)P\big)\\ &= \det\big(P^{-1}\big)\det(xI-A)\det(P)=\det(xI-A)=c_{A}(x). \end{aligned} \] We conclude from the above equation that $A$ and $B$ have the same characteristic polynomial, and since the eigenvalues of a matrix coincide with the roots of the characteristic polynomial, we also conclude that $A$ and $B$ have the same eigenvalues.

The converse of the above theorem is not true. For instance, the matrices \[ I= \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix},\qquad B= \begin{bmatrix} 1 & 1 \\ 0 & 1 \end{bmatrix} \] have the same determinant, trace, characteristic polynomial and eigenvalues, but they are not similar, because if $P$ is any invertible matrix we have $P^{-1}IP=I\neq B$.

Chapter 6: Vector Spaces

6.1 Examples and Basic Properties

The purpose of this chapter is to introduce the theory of vector spaces. We have already introduced the most important example of a vector space, which is $\RR^n$. However, there are other mathematical objects that behave similarly to vectors, and that fact will allow us to also do linear algebra with them.

The introduction of general vector spaces is an example of abstraction, which is a very common technique in mathematical reasoning. By applying intuition that we have with vectors in $\RR^n$ to other mathematical entities, we are able to obtain a new perspective on said entities and also obtain new results.

A vector space is a triple $(V,+,\cdot)$ consisting of three pieces of data:
  • A nonempty set $V$, whose elements are called vectors.
  • A map $+$ that takes any pair $(\bf{v},\bf{w})$ consisting of elements of $V$ and returns another element $\bf{v}+\bf{w}\in V$.
  • A map $\cdot$ that takes any scalar $a\in \RR$ and any vector $\bf{v}\in V$ and returns a vector $a\bf{v}\in V$.
The operation $+$ is called vector addition, whereas $\cdot$ is called scalar multiplication. In addition, both operations are going to be subject to the following relations:
  1. $\bf{u}+\bf{v}=\bf{v}+\bf{u}$ for all $\bf{u},\,\bf{v}\in V$.
  2. $\bf{u}+(\bf{v}+\bf{w})=(\bf{u}+\bf{v})+\bf{w}$ for all $\bf{u},\,\bf{v},\,\bf{w}\in V$.
  3. There exists an element $\bf{0}\in V$, called the zero vector, such that $\bf{v}+\bf{0}=\bf{0}+\bf{v}=\bf{v}$ for all $\bf{v}\in V$.
  4. For each $\bf{v}\in V$, there exists another element $-\bf{v}\in V$ that satisfies $\bf{v}+(-\bf{v})=-\bf{v}+\bf{v}=\bf{0}$.
  5. $a(\bf{v}+\bf{w})=a\bf{v}+a\bf{w}$ for all $a\in \RR$ and $\bf{v},\,\bf{w}\in \RR$.
  6. $(a+b)\bf{v}=a\bf{v}+b\bf{v}$ for all $a,\,b\in \RR$ and $\bf{v}\in\RR$.
  7. $(ab)\bf{v}=a(b\bf{v})$ for all $a,\,b\in \RR$ and $\bf{v}\in V$.
  8. $1\bf{v}=\bf{v}$ for all $\bf{v}\in V$.

For simplicity, we usually denote a vector space $(V,+,\cdot)$ only by its underlying set $V$.

$\RR^n$ is a vector space when endowed with the vector addition and scalar multiplication that were defined in Chapter 1.
Suppose $U\subseteq \RR^n$ is a subspace of $\RR^n$. Then $U$ is also a vector space itself when considered with the same vector addition and scalar multiplication as $\RR^n$.
Let $m$ and $n$ be two natural numbers and consider the set \[ M_{mn}=\{ X \mid X \text{ is an $m\times n$ matrix} \} . \] Then $M_{mn}$ is a vector space when endowed with the vector addition and scalar multiplication that were defined in Chapter 2.

We now introduce the vector space of real polynomials. To do this, we need to remind some concepts regarding polynomials.

A polynomial $p(x)$ is a function of the form \[ p(x)=a_{n}x^{n}+\cdots+a_{1}x+a_{0}, \] where $a_{0},\,\dots,\,a_{n}\in \RR$ are real constants called, the coefficients of $p(x)$.

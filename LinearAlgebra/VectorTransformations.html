Chapter 7: Linear Transformations

7.1 Examples and Elementary Properties

In this chapter we study linear transformations between (abstract) vector spaces.

Linear Transformations and Examples

Just as how we defined linear transformations $T\colon \RR^n\to \RR^m$, we can define linear transformations between vector spaces.

Let $V$ and $W$ be two vector spaces. We say that a transformation $T\colon V \to W$ is a linear transformation if it satisfies the following two conditions:
  1. $T(\bf{x}+\bf{y})=T(\bf{x})+T(\bf{y})$ for all $\bf{x},\,\bf{y}\in V$ (we say that $T$ preserves vector addition).
  2. $T(a\bf{x})=aT(\bf{x})$ for all $\bf{x}\in V$ and $a\in \RR$ (we say that $T$ preserves scalar multiplication).

Let us go through some examples of linear transformations.

If $V$ is any vector space, then the following transformations are linear:
  1. The identity transformation $1_V\colon V\to V$ given by $1_V(\bf{v})=\bf{v}$.
  2. The zero transformation $0\colon V\to V$ given by $0(\bf{v})=\bf{0}$.
  3. The scalar transformation $a\colon V\to V$ (where $a\in\RR$) given by $a(\bf{v})=a\bf{v}$.
As we already saw in Section 2.6, the linear transformations $T\colon\RR^n\to\RR^m$ are precisely the matrix transformations. In other words, if $T\colon\RR^n\to\RR^m$ is linear, then $T=T_A$ for some $m\times n$ matrix $A$.
The transposition map $R\colon \bf{M}_{mn}\to\bf{M}_{nm}$ given by $R(A)=A^T$ is linear (because of the properties of matrix transposes).
The trace $\operatorname{tr}\colon \bf{M}_{nn}\to\RR$ is a linear transformation. Recall that the trace of a matrix was defined by \[ \tr(A)=a_{11}+\cdots+a_{nn}, \quad \textsf{where } A=[a_{ij}]. \]
Proof that $\tr$ is linear.

Suppose $A=[a_{ij}]$ and $B=[b_{ij}]$ are two $n\times n$ matrices. Then we have \[ \begin{aligned} \tr(A+B) &= \tr[a_{ij}+b_{ij}] = (a_{11}+b_{11})+\cdots+(a_{nn}+b_{nn})\\[1em] &= (a_{11}+\dots+a_{nn})+\cdots+(b_{11}+\cdots+b_{nn}) = \tr(A)+\tr(B). \end{aligned} \]

Similarly, if $A=[a_{ij}]$ is in $\bf{M}_{nn}$ and $a\in\RR$, we have \[ \tr(aA) = \tr[aa_{ij}] = aa_{11}+\cdots+aa_{nn} = a(a_{11}+\cdots+a_{nn}) = a\tr(A). \]

We conclude that $\tr$ is a linear transformation.

The determinant $\det\colon\bf{M}_{nn}\to\RR$ is not linear (except in the case $n=1$).
Suppose $a\in \RR$ is a scalar. Then we can define two linear transformations on polynomials:
  • The evaluation (at $a$) $E_a\colon\bf{P}_n\to\RR$ given by $E_a(p(x))=p(a)$. Explicitly: \[ E_a(a_0+a_1x+a_2x^2+\cdots+a_nx^n)= a_0+a_1a+a_2a^2+\cdots+a_na^n. \]
  • The shift (by $a$) $S_a\colon \bf{P}_n\to\bf{P}_n$ given by $S_a(p(x))=p(x+a)$. Explicitly: \[ S_a(a_0+a_1x+a_2x^2+\cdots+a_nx^n)= a_0+a_1(x+a)+a_2(x+a)^2+\cdots+a_n(x+a)^n. \]
Proof that $E_a$ and $S_a$ are linear.

Firstly, assume $p(x)$ and $q(x)$ are two polynomials in $\bf{P}_n$. Then we have \[ \begin{aligned} E_a((p+q)(x)) &= (p+q)(a) = p(a)+q(a) = E_a(p(x))+E_a(q(x)), \\[1em] S_a((p+q)(x)) &= (p+q)(x+a) = p(x+a)+q(x+a) = S_a(p(x))+S_a(q(x)). \end{aligned} \]

Similarly, if $p(x)\in\bf{P}_n$ and $c\in \RR$, then we get \[ \begin{aligned} E_a((cp)(x)) &= (cp)(a) = cp(a) = cE_a(p(x)),\\[1em] S_a((cp)(x)) &= (cp)(x+a) = cp(x+a) = cS_a(p(x)). \end{aligned} \]

Therefore, both $E_a$ and $S_a$ are linear transformations.

The derivative $D\colon\bf{P}_n\to\bf{P}_{n-1}$ given by $D(p(x))=p'(x)$ is a linear transformation.
Proof that $D$ is linear.

Suppose $p(x)$ and $q(x)$ are two polynomials of degree at most $n$. Then we have \[ D((p+q)(x)) = (p+q)'(x) = p'(x)+q'(x) = D(p(x))+D(q(x)). \]

In addition, if $p(x)\in\bf{P}_n$ and $a\in\RR$ is a constant, we see that \[ D((ap)(x)) = (ap)'(x) = ap'(x) = aD(p(x)). \]

In conclustion, $D$ is a linear transformation.

Properties of Linear Transformations

We now explore some of the main features of linear transformations. More importantly, we will see that it is possible to completely determine a linear transformation $T\colon V \to W$ by looking only at the images of a basis of $V$.

Suppose $T\colon V\to W$ is any linear transformation. Then the following identities are true:
  1. $T(\bf{0})=\bf{0}$.
  2. If $\bf{x}\in V$, then $T(\bf{x})=-\bf{x}$.
  3. If $\bf{x}_1,\,\bf{x}_2,\,\dots,\,\bf{x}_k$ are vectors in $V$ and $a_1,\,a_2,\,\dots,\,a_k\in\RR$ are scalars, then \[ T(a_1\bf{x}_1+a_2\bf{x}_2+\cdots+a_k\bf{x}_k) = a_1T(\bf{x}_1)+a_2T(\bf{x}_2)+\cdots+a_kT(\bf{x}_k). \]
Proof.

The proof of the third equation is exactly the same as in the case of linear transformations $T\colon\RR^n\to\RR^m$. Thus, we only have to focus on the first two identities.

For the first identity, note that \[ T(\bf{0})=T(0\,\bf{0})=0\,T(\bf{0})=\bf{0}. \] For the second one, we can prove it as follows: \[ T(-\bf{v})=T((-1)\bf{v})=(-1)T(\bf{v})=-T(\bf{v}). \]

The last identity on the previous theorem must be understood as follows. Suppose we are given a linear transformation $T\colon V\to W$ and vectors $\bf{x}_1,\,\bf{x}_2,\,\dots,\,\bf{x}_k\in V$. If we know the images $T(\bf{x}_1),\,T(\bf{x}_2),\,\dots,\,T(\bf{x}_k)$, then we can automatically determine $T(\bf{v})$ when $\bf{v}$ is a linear combination of the vectors $\bf{x}_1,\,\bf{x}_2,\,\dots,\,\bf{x}_k$ (that is, when $\bf{v}$ is in the subspace $\vecspan\{\bf{x}_1,\bf{x}_2,\dots,\,\bf{x}_k\}$).

Let's make this idea more precise. Suppose that $T,\,S\colon V\to W$ are transformations (they do not have to be linear). Then we say that $T$ and $S$ are equal and we write $T=S$ if $T(\bf{x})=S(\bf{x})$ for all $\bf{x}\in V$.

Having this in mind, we obtain the following result:

Let $T,\,S\colon V\to W$ be two linear transformations and suppose that $\{\bf{x}_1,\,\dots,\,\bf{x}_k\}$ is a spanning set for $V$. If $T(\bf{x}_i)=S(\bf{x}_i)$ for all $i$, then $T=S$.
Proof.

Let $\bf{x}$ be any vector in $V$. We need to show that $T(\bf{x})=S(\bf{x})$.

Since $\bf{x}\in V =\vecspan\{\bf{x}_1,\dots,\bf{x}_k\}$, we can write \[ \bf{x}=a_1\bf{x}_1+\cdots+a_k\bf{x}_k. \] Thus, applying $T$ we see that \[ T(\bf{x}) = a_1T(\bf{x}_1)+\cdots+a_kT(\bf{x}_k) = a_1S(\bf{x}_1)+\cdots+a_kS(\bf{x}_k) = S(\bf{x}). \]

Because $\bf{x}$ was any vector in $V$, we deduce that $T=S$.

Now we present a result on the existence and uniqueness of linear transformations by means on how they act on a basis.

Suppose $V$ and $W$ are vector spaces, and let $\{\bf{e}_1,\dots,\bf{e}_n\}$ be a basis of $W$. If $\bf{w}_1,\,\dots,\,\bf{w}_n\in W$ are vectors in $W$ (they need not be a spanning set or independent), then there exists a unique linear transformation $T\colon V\to W$ satisfying $T(\bf{e}_i)=\bf{w}_i$ for all $i$.
Proof.

Let us first prove that the transformation $T$ exists by explicitly defining it. Suppose $\bf{x}\in V$ is any vector. Then, since $\{\bf{e}_1,\dots,\bf{e}_n\}$ is a basis of $V$, there exist unique constants $a_1,\,\dots,\,a_n\in\RR$ that satisfy \[ \bf{x}=a_1\bf{e}_1+\cdots+a_n\bf{e}_n. \] Then, we define $T(\bf{x})$ by the equation \[ T(\bf{x})=a_1\bf{w}_1+\cdots+a_n\bf{w}_n. \] Since the constants $a_1,\,\dots,\,a_n$ are unique, the above definition can be made without any risk of ambiguity. Now, it is clear that $T(\bf{e}_i)=\bf{w}_i$ for all $i$ because of how the transformation $T$ was defined.

To show uniqueness, suppose that $S\colon V\to W$ is another linear transformation such that $S(\bf{x}_i)=\bf{w}_i=T(\bf{x}_i)$ for all $i$. Since a basis of $V$ is in particular a spanning set, we obtain that $S=T$ by the preceding theorem. therefore, $T$ is unique.

The existence and uniqueness theorem asserts that in order to define any linear transformation $T\colon V \to W$, it suffices to determine the image under $T$ of the elements in any basis of our choice.

The set $\{1+x,2x-x^2,3x^2\}$ is a basis of $\bf{P}_2$. Find a linear transformation $T\colon \bf{P}_2\to \RR^2$ that satisfies \[ T(1+x)=(0,2),\qquad T\big(2x-x^2\big)=(-2,7),\qquad T\big(3x^2\big)=(0,-9). \]
Proof.

To find $T$, we need to know the image of any polynomial under $T$.

Let $p(x)=a+bx+cx^2$ be a polynomial in $\bf{P}_2$. We have to write $p(x)$ as a linear combination of $1+x$, $2x-x^2$ and $3x^2$. To do this, we write \[ a+bx+cx^2 = \alpha(1+x) + \beta\big(2x-x^2\big) + \gamma\big(3x^2\big) = \alpha + (\alpha+2\beta)x + (-\beta+3\gamma)x^2, \] where $\alpha,\,\beta,\,\gamma\in\RR$ are constants to be determined. From the equation above, we see that these constant satisfy the system \[ \left\{ \begin{array}{ccccccc} \alpha & & & & & = & a \\ \alpha & + & 2\beta & & & = & b \\ & - & \beta & + & 3\gamma & = & c \end{array} \right. \] We see that the only solution to the above system is \[ \alpha = a, \qquad \beta = \frac{b-a}{2}, \qquad \gamma = \frac{-a+b+2c}{6}. \] So we have that \[ p(x) = a(1-x) + \frac{b-a}{2}\big(2x-x^2\big) + \frac{-a+b+2c}{6}\big(3x^2\big). \]

Then, we deduce from the existence and uniqueness theorem that \[ T(p(x)) = a(0,2) + \frac{b-a}{2}(-2,7) + \frac{-a+b+2c}{6}(0,-9) =(a-b,2b-3c). \]

In conclusion, the transformation $T$ is given by \[ T\big(a+bx+cx^2\big) = (a-b,2b-3c). \]

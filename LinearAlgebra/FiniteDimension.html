6.4 Finite Dimensional Vector Spaces

In this section, we continue our exploration of linearly independent subsets and bases of vector spaces.

We have already seen examples of bases in some vector spaces. For example:

  • $\{\bf{e}_1,\dots,\bf{e}_n\}$ is a basis of $\RR^n$.
  • $\{E_{11},E_{12},\dots,E_{mn}\}$ is a basis of $\bf{M}_{mn}$.
  • $\{1,x,x^2,\dots,x^n\}$ is a basis of $\bf{P}_n$.

However, in general, a vector space may not have a basis. In fact, this will be the case for $\bf{P}$ and $\bf{F}[a,b]$. However, for the purposes of this course we will only worry about understanding vector spaces that do have a basis.

Finite Dimensional Spaces

Let $V$ be a vector space. We say that $V$ is finite dimensional if it admits a (finite) spanning set. Otherwise, we say that $V$ is infinite dimensional.

We start by giving some conditions that characterize when a vector space is finite dimensional.

If $V$ is a vector space, then the following conditions are equivalent:
  1. $V$ is finite dimensional.
  2. $V$ has a (finite) basis.
  3. There exists an integer $n\geq 0$ such that any subset of $V$ with more than $n$ vectors is linearly dependent.
Proof.

We first prove that 1 implies 2. Assume that $V$ is finite dimensional, so it contains some spanning set $\{\bf{v}_1,\dots,\bf{v}_m\}$. We will show that this spanning set contains a basis of $V$.

If $\{\bf{v}_1,\dots,\bf{v}_m\}$ is already a basis of $V$, then we are done. Otherwise, there exists some linear combination \[ a_1\bf{v}_1 + \cdots + a_m \bf{v}_m = \bf{0} \] where at least one of the coefficients in the combination is nonzero. After relabelling the vectors if necessary, we can suppose that $a_1\neq 0$. Then we can see that \[ \bf{v}_1 = -\frac{a_2}{a_1}\bf{v}_2 - \cdots - \frac{a_m}{a_1}\bf{v}_m\in \vecspan\{\bf{v}_2,\dots,\bf{v}_m\}, \] which implies that \[ V=\vecspan\{\bf{v}_1,\dots,\bf{v}_m\}=\vecspan\{\bf{v}_2,\dots,\bf{v}_m\}. \] Now, if $\{\bf{v}_2,\dots,\bf{v}_m\}$ is linearly independent, then it is a basis, and we are done. If not, we can repeat the argument above to see that (after perhaps relabelling the vectors) \[ V=\vecspan\{\bf{v}_3,\dots,\bf{v}_m\}. \] We can continue this process, deleting vectors from our original spanning set appropriately, until we reach a basis. Therefore, 2 is true.

Next, we show that 2 implies 3. If we take $n=\dim V$ (which is possible because $V$ has a basis by assumption), then the Steinitz Exchange Lemma implies that any set of vectors in $V$ with more than $n$ elements is linearly dependent. This proves our claim.

Finally, let us check that 3 implies 1. Since we are assuming that 3 holds, we can find a natural number $n$ such that there exists a linearly independent set $\{\bf{v}_1,\dots,\bf{v}_n\}$ in $V$ but any subset of $V$ with more than $n$ elements is not independent. We claim that $\{\bf{v}_1,\dots,\bf{v}_n\}$ is a spanning set (in fact, a basis) of $V$.

To show this, take any vector $\bf{v}\in V$. The set $\{\bf{v},\bf{v}_1,\dots,\bf{v}_n\}$ is linearly dependent because it has $n+1$ elements, so we can find a linear combination \[ a\bf{v}+b_1\bf{v}_1+\dots+b_n\bf{v}_n=\bf{0} \] where one of the coefficients is nonzero. Actually, we need to have $a=0$, since otherwise we would get \[ b_1\bf{v}_1+\dots+b_n\bf{v}_n=\bf0{} \] with one of the coefficients being nonzero, and this contradicts linear independence of $\{\bf{v}_1,\dots,\bf{v}_n\}$. Therefore, because $a\neq 0$, we get \[ \bf{v}=-\frac{b_1}{a}\bf{v}_1-\cdots-\frac{b_n}{a}\bf{v}_n\in\vecspan\{\bf{v}_1,\dots,\bf{v}_n\}. \] Since $\bf{v}$ is any vector, we conclude that $V=\vecspan\{\bf{v}_1,\dots,\bf{v}_n\}$, so $V$ has a finite spanning set, thus showing 1.

The above theorem, together with the Steinitz exchange lemma, has several important consequences. The first one gives us some hints on how to find bases of vector spaces, and it extends what we already know from the case of $\RR^n$:

Let $V$ be a finite dimensional vector space and $n=\dim V$. Then the following statements are true:
  1. Every spanning set of $V$ contains a basis.
  2. If $B$ is a basis of $V$ and $C$ is a linearly independent subset of $V$, then $C$ can be enlarged to a basis of $V$ by appropriately adding elements of $B$.
Proof.

The first statement is actually proved when showing that 1 implies 2 in the characterization of finite dimensional spaces. Thus, we just have to show the second statement.

Assume $B=\{\bf{v}_1,\dots,\bf{v}_n\}$ is a basis of $V$ and $C=\{\bf{u}_1,\dots,\bf{u}_k\}$ is a linearly independent set. By the Steinitz exchange lemma we know that $k\leq n$ and (after perhaps changing the order of the elements in $B$), the set $C'=\{\bf{u}_1,\dots,\bf{u}_k,\bf{v}_{k+1},\dots,\bf{v}_n\}$ is a spanning set for $V$. The set $C'$ is also a basis. Indeed, if $C'$ were not a basis then by part 1 it would contain a basis with less than $n$ elements, contradicting the fact that $\dim V = n$. Therefore, 2 is also true.

As a consequence of the above theorem, we recover a result that can be proved with the same arguments as its analogue in $\RR^n$.

Suppose $V$ is a finite dimensional vector space with $\dim V = n$ and $B=\{\bf{v}_1,\dots,\bf{v}_n\}$ is a subset of $V$ with exactly $n$ elements. Then the following conditions are equivalent:
  1. $B$ spans $V$.
  2. $B$ is linearly independent.
  3. $B$ is a basis of $V$.
The subset $\{1-2x,5x^2\}$ is linearly independent in $\bf{P}_2$. Find a basis of $\bf{P}_2$ containing this set.
Solution.

We know that the set $\{1,x,x^2\}$ is a basis of $\bf{P}_2$, so there must exist an element in this basis that can be added to $\{1-2x,5x^2\}$ in order to obtain a basis of $\bf{P}_2$. For instance, we can try adding the polynomial $1$ to obtain the set $\{1-2x,5x^2,1\}$. This set is linearly independent (check this as an exercise!), and because $\dim \bf{P}_2 =3$ it follows that $\{1-2x,5x^2,1\}$ is a basis of $\bf{P}_2$.

If you consider adding $x$ instead of $1$ in the above exercise, that will also yield a basis. However, adding $x^2$ will not result in a basis.
Suppose $\bf{c}_1,\,\dots,\,\bf{c}_k$ are linearly independent vectors in $\RR^n$. Prove that there exists an invertible $n\times n$ matrix $A$ whose first $k$ rows are $\bf{c}_1,\,\dots,\,\bf{c}_k$.
Solution.

Since $\{\bf{c}_1,\dots,\bf{c}_k\}$ is a linearly independent subset of $\RR^n$ and $\dim \RR^n = n$, we can find vectors $\bf{c}_{k+1},\,\dots,\,\bf{c}_n\in \RR^n$ that satisfy that $\{\bf{c}_1,\dots,\bf{c}_k,\bf{c}_{k+1},\dots,\bf{c}_n\}$ is a basis of $\RR^n$. Consequently, the matrix \[ A= \begin{bmatrix} \bf{c}_1 & \cdots & \bf{c}_k & \bf{c}_{k+1} & \cdots & \bf{c}_n \end{bmatrix} \] is an invertible $n\times n$ matrix whose first $k$ columns are $\bf{c}_1,\,\dots,\,\bf{c}_k$, as desired.

Subspaces of Finite Dimensional Spaces

As one may expect, a subspace of a finite dimensional vector space is automatically a finite dimensional vector space in its own right:

Let $V$ be a finite dimensional vector space and $U\subseteq V$ a vector subspace. Then:
  1. $U$ is finite dimensional and $\dim U \leq \dim V$.
  2. $U=V$ if and only if $\dim U = \dim V$.
Proof.

Because of the characterization of finite dimensional spaces, to show that $U$ is finite dimensional it is enough to prove that there exists some natural number $n$ such that any subset of $U$ with more than $n$ elements is dependent.

Set $n=\dim V$. Then any subset of $V$ with more than $n$ vectors is linearly dependent. Now, $U$ is contained in $V$, so any subset of $U$ with more than $n$ vectors is automatically dependent. Becaues of this, we conclude that $U$ is finite dimensional.

Let us show that $\dim U \leq n = \dim V$. If $B=\{\bf{v}_1,\dots,\bf{v}_k\}$ is a basis of $U$, then in particular it is a linearly independent subset of $V$. By the Steinitz exchange lemma, this means that $\dim U = k \leq n = \dim V$, so our claim follows.

Let us prove the second assertion. If $\dim U = \dim V = n$ and $B$ is a basis of $U$, then it is also a linearly independent subset of $V$ having $n$ elements, so it must be a basis of $V$ as well because $\dim V = n$. This means that $U=\vecspan B = V$, as desired.

Because of the above theorem, we have that if $U$ is a subspace of a finite dimensional vector space $V$, then

  • $U=\{\bf{0}\}$ if and only if $\dim U = 0$.
  • $U=V$ if and only if $\dim U = \dim V$.

Consequently, a proper subspace of $V$ will always have dimension between $1$ and $\dim V - 1$.

The previous results can be used to minimize computations in solving exercises. Here are some examples.

Let $a\in \RR$ be a fixed real number, and consider the vector subspace \[ V=\{p(x)\in\bf{P}_n \mid p(a)=0 \}\subseteq \bf{P}_n. \] Show that $\dim V=n$ and find a basis of $V$.
Solution.

Since $V$ is a subspace of $\bf{P}_n$ and $V\neq \bf{P}_n$, we have $\dim V \leq \dim\bf{P}_n - 1=n$. Moreover, if \[ p(x)=c_0 + c_1 x + c_2 x^2 + \cdots + c_n x^n \] is a polynomial in $\bf{P}_n$, then $p(x)\in V$ if and only if the coefficients of $p(x)$ satisfy \[ c_0 + c_1 a + c_2 a^2 + \cdots + c_n a^n = 0. \] This implies that, for instance, the polynomials \[ x-a, \,x^2-a^2,\,\dots,\,x^n-a^n \] are all in $V$. These polynomials are also linearly independent. Indeed, if we have a linear combination \[ u_1(x-a)+u_2(x^2-a^2)+\cdots+u_n(x^n-a^n)=0, \] then equating the coefficients of the polynomials on each side we obtain $u_1=\dots=u_n=0$, showing that the set \[\{x-a,x^2-a^2,\dots,x^n-a^n\}\] is a linearly independent subset of $\bf{P}_n$. Since this set has $n$ elements, it follows that $\dim V \geq n$. However, $\dim V \leq n$ from what we saw before, so $\dim V=n$ and \[ \{x-a,x^2-a^2,\dots,x^n-a^n\} \] is a basis of $V$.

It is worth mentioning that there are other natural choices for bases of $V$, such as \[ \{x-a,(x-a)^2,\dots,(x-a)^n\} \] or \[ \{x-a,x(x-a),x^2(x-a),\dots,x^{n-1}(x-a)\}. \] You can try to check that these sets are bases, for instance, showing that they are linearly independent.

The set \[ \{3+2x+4x^2, 1+x+x^2, 5-x-2x^2, 2x-3x^2\} \] is a spanning set for $\bf{P}_2$. Find a basis of $\bf{P}_2$ inside of this set.
Solution.

Because $\dim \bf{P}_2 = 3$, we know that there must exist a polynomial in the above set such that the result of removing it is a basis of $\bf{P}_2$. Let us try to find one such polynomial.

For instance, if we remove the polynomial $3+2x+4x^2$, we are left with the set \[ B = \{1+x+x^2,5-x-2x^2,2x-3x^2\}. \] It can be shown directly that $B$ is linearly independent, and since $\dim \bf{P}_2=3$, we conclude that $B$ is a basis of $\bf{P}_2$.

Show that the vector space $\bf{P}$ is not finite dimensional.
Solution.

Recall that $\bf{P}_n\subseteq \bf{P}$ for every $n=1,2,\dots$. In particular, if $\bf{P}$ were a finite dimensional vector space (say $\dim \bf{P}=m$), then we would have $m=\dim \bf{P}\geq \dim \bf{P}_n = n+1$ for every $n$. In particular, by choosing $n=m$ we see that $m\geq m+1$, a contradiction. Therefore, $\bf{P}$ is infinite dimensional.

We finish this section with a formula to compute the dimension of the sum of two vector spaces. If $V$ is a vector space and $U,\,W\subseteq V$ are vector subspaces of $V$, then the sum and intersection of $U$ and $V$ are defined to be the subspaces \[ \begin{aligned} U+V &= \{\bf{u}+\bf{w}\mid\bf{u}\in U,\,\bf{w}\in W\}.\\[1em] U\cap V &= \{\bf{v}\mid\bf{v}\in U \textsf{ and } \bf{v}\in W\}. \end{aligned} \]

Let $V$ be a finite dimensional vector space and $U,\,W\subseteq V$ two subspaces. Then \[ \dim U + \dim V = \dim (U+V) + \dim (U\cap V). \]
Proof.

Let us first define $k=\dim(U\cap W)$, $n=\dim U$ and $m=\dim W$. Our objective is to prove that $\dim(U+W)=m+n-k$.

We start with a basis $\{\bf{x}_1,\dots,\bf{x}_k\}$ of $U\cap W$. On one hand, since $\{\bf{x}_1,\dots,\bf{x}_k\}$ is linearly independent in $U$, there exist vectors $\bf{y}_1,\,\dots,\,\bf{y}_{n-k}\in U$ such that $\{\bf{x}_1,\dots,\bf{x}_k,\bf{y}_1,\dots,\bf{y}_{n-k}\}$ is a basis of $U$. On the other hand, for the exact same reason there exist vectors $\bf{z}_1,\,\dots,\,\bf{z}_{m-k}\in W$ such that $\{\bf{x}_1,\dots,\bf{x}_k,\bf{z}_1,\dots,\bf{z}_{m-k}\}$ is a basis of $W$.

We claim that the set \[ \{\bf{x}_1,\dots,\bf{x}_k,\bf{y}_{1},\dots,\bf{y}_{n-k},\bf{z}_1,\dots,\bf{z}_{m-k}\} \] is a basis of $U+W$. From the definition of $U+W$, it is clear that the above set is a spanning set for $U+W$. We thus are left with showing that it is linearly independent.

Let us suppose that we have a linear combination of the above vectors that yields zero: \[ a_1\bf{x}_1+\cdots+a_k\bf{x}_k + b_1\bf{y}_1+\cdots+b_{n-k}\bf{y}_{n-k} + c_1\bf{z}_1+\cdots+c_{m-k}\bf{z}_{m-k} = \bf{0}. \] We have to prove that all the coefficients in this combination vanish. To do this, observe that the above equation implies that \[ a_1\bf{x}_1+\cdots+a_k\bf{x}_k + b_1\bf{y}_1+\cdots+b_{n-k}\bf{y}_{n-k} = -c_1\bf{z}_1-\cdots-c_{m-k}\bf{z}_{m-k}. \]

Let $\bf{v}$ be the vector that is equal to any of the sides of the above equation. The expression in the left hand side shows that $\bf{v}\in U$, whereas the expression in the right hand side says that $\bf{v}\in W$. Therefore, $\bf{v}\in U\cap W$, so it has to be a linear combination of the $\bf{x}_i$. As $\{\bf{x}_1,\dots,\bf{x}_k,\bf{y}_1,\dots,\bf{y}_{n-k}\}$ is linearly independent, this yields $b_1=\cdots=b_{n-k}=0$. Thus, the first equation becomes \[ a_1\bf{x}_1+\cdots+a_k\bf{x}_k + c_1\bf{z}_1+\cdots+c_{m-k}\bf{z}_{m-k} = \bf{0}, \] and because $\{\bf{x}_1,\dots,\bf{x}_k,\bf{z}_1,\dots,\bf{z}_{m-k}\}$ is linearly independent it follows that $a_1=\cdots=a_k=c_1=\cdots=c_{m-k}$. We conclude that the set in question is linearly independent, and thus a basis of $U+W$.

To conclude, note that the dimension of $U+W$ is the number of elements in the basis, namely, \[ \dim (U+V)=k+(n-k)+(m-k)=n+m-k=\dim U + \dim W - \dim (U\cap W). \] This finishes the proof.
